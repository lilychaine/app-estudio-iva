# 📚 IVA Study App — Test interactivo de Fiscalidad

Aplicación web simple para **practicar preguntas tipo test de IVA español** basadas en normativa real.

La app está diseñada como **herramienta de estudio** para estudiantes de:

- Derecho
- ADE
- Fiscalidad
- Opositores a Hacienda
- Máster en Tributación

Permite entrenar conceptos clave del **Impuesto sobre el Valor Añadido (IVA)** mediante preguntas tipo examen.

---

# 🎯 Objetivo del proyecto

Este proyecto nace para resolver un problema muy común al estudiar fiscalidad:

> Memorizar la normativa del IVA sin practicar preguntas tipo examen.

La aplicación permite:

- practicar preguntas tipo test
- detectar lagunas de conocimiento
- repasar normativa de forma activa
- simular tests de examen

Todo en una **interfaz simple pensada para estudiar**.

---

# 🧠 Contenido académico

Las preguntas están basadas en:

- Ley del IVA (LIVA)
- Reglamento del IVA (RIVA)
- Reglamento de Facturación
- Apuntes académicos de IVA

Los temas cubiertos incluyen:

- Naturaleza del IVA
- Ámbito territorial
- Hecho imponible
- Empresario o profesional
- Entregas de bienes
- Prestaciones de servicios
- Adquisiciones intracomunitarias
- Importaciones
- Devengo
- Base imponible
- Sujeto pasivo
- Deducciones
- Regímenes especiales
- Recargo de equivalencia
- Obligaciones de facturación
- Factura simplificada
- Factura electrónica
- Facturas rectificativas
- Conservación de facturas

Cada pregunta incluye:

- 4 opciones (A B C D)
- 1 respuesta correcta
- explicación
- referencia normativa

---

# ⚙️ Funcionalidades

### 📊 Test interactivo

- preguntas tipo test
- respuesta inmediata
- explicación jurídica

### 🧩 Modos de estudio

- modo secuencial
- modo aleatorio
- simulacro de examen (20 preguntas)

### 📈 Seguimiento de progreso

La aplicación muestra:

- número de aciertos
- número de errores
- porcentaje de acierto
